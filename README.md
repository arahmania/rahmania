# 🚀 Portfolio

Personal developer portfolio built with [Astro](https://astro.build)

## 🧞 Commands

Jalankan dari root project:

| Command                   | Action                                      |
| :------------------------ | :------------------------------------------ |
| `npm install`             | Install semua dependencies                  |
| `npm run dev`             | Dev server di `localhost:4321`              |
| `npm run build`           | Build production ke `./dist/`              |
| `npm run preview`         | Preview hasil build secara lokal            |
| `npm run astro ...`       | Jalankan Astro CLI commands                 |
| `npm run astro -- --help` | Lihat semua opsi Astro CLI                  |

## 📦 Getting Started

```sh
git clone <repo-url>
cd portfolio
npm install
npm run dev
```
